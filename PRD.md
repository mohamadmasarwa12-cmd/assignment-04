# PRD.md

## Project Title

Creating PDF Documents Using LaTeX

## Purpose

The purpose of this project is to demonstrate the creation of professional PDF documents using LaTeX and LuaLaTeX.

## Goals

* Create an academic PDF document.
* Use a custom CLS file.
* Demonstrate tables, figures, graphs, and references.
* Generate a final PDF using LuaLaTeX.
* Organize project files in a structured repository.

## Functional Requirements

* The document must contain at least 50 English words.
* The document must include a table.
* The document must include a figure.
* The document must include a graph created with TikZ/PGFPlots.
* The document must include a bibliography section.
* The document must compile successfully into PDF.

## Deliverables

* main.tex
* academic-template.cls
* main.pdf
* GitHub repository documentation
